## Hangman game

### Installation

1. to install all the dependencies `pnpm install`
2. create `.env` at the root of the repo
3. add `BE_PORT="3030"` in the `.env``
4. to run the project `pnpm dev`

**Tech stack**

- _Monorepo_

1. TurboRepo
2. pnpm

- _Frontend_

1. Next.js
2. Zustand

4. Framer motion
